Responsive image and width

sizes attribute: Lists which size image to use at which media query

srcset attribute: Allows you to specify multiple files to be used based on viewport width or image pexel width

Responsive picture

Nest img, source, and srcset information inside of a picture element to create an image that changes content depending on the surrounding layout

The srcset attribute lets the browser choose from options that fit the specific situation, minimizing bandwidth consumption while producing attractive results.

Captions and subtities

To get these captions to show up on our video we put a track element inside the video element.

On the track element:
use the source attribute to point to the file itself.
use the kind attribute to tell the browser that this is captions.
add a label of english that will show up in the player as a label for this choice.
use a source lang attribute to specify which language this is.
put a default attribute on this track element to specify that this track is the one we want to use by default when a user turns on captions
